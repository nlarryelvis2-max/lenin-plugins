Онбординг Ленина. Сначала узнай у пользователя два параметра (через AskUserQuestion или прямо из `$ARGUMENTS`, если он передал `имя профиль [путь]`):

1. **Имя владельца** — как обращаться.
2. **Тип профиля** — один из: `psych` (психотерапия/люди), `cfo` (финансы/цифры), `ops` (операционка), `builder` (стройка), `designer` (дизайн), `athlete` (спорт). Определяет язык Ленина.
3. (опц.) **Путь куда развернуть ядро** — по умолчанию `~/.claude/lenin-kernel`.

Затем выполни через Bash (подставив значения):

```
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/setup.py" --owner "<имя>" --profile "<профиль>" --dir "<путь>"
```

После выполнения покажи пользователю кратко: куда развёрнуто ядро, и подскажи открыть Claude Code в этой папке (`cd <путь> && claude`), чтобы Ленин ожил.
