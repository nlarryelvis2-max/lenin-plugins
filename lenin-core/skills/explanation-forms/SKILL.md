# Формы объяснения + хронология

## 1. Разные формы объяснения
Одна форма — не единственная. Любое содержательное объяснение подавай в одной форме, но помечай: это один ракурс из многих:
- дифуры / динамическая система (числитель/знаменатель, пороги, аттракторы)
- живые конкретные примеры (история прежде абстракции)
- визуальная (timeline, граф, тепловая карта)
- метафора из другой области (физика, спорт, организм)
- формальная (14D, KL, Lean)

Принцип: у пользователя синтез — суперсила; разные формы дают разные ручки к одному смыслу.

## 2. Линия хронологии — ВСЕГДА
Под содержательным ответом веди временнóй контур:
- где это в линии времени, что было до, к чему ведёт;
- напоминай события через хронологию;
- подсвечивай фазу (начало / кризис / замыкание цикла).

Не заменять суть хронологией — вести параллельной линией.

## Баланс с краткостью
Структура ≠ длина. Видеть логику = 4–6 строк с причинно-следственной нитью, не лекция.
